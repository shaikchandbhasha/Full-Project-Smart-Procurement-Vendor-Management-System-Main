# Smart Procurement Vendor Management System

## 📌 Description
A web-based application to manage vendors, procurement processes, purchase orders, invoices, and payments efficiently.

## 🚀 Features
- Vendor registration & management
- Purchase order creation
- Invoice & payment tracking
- Role-based dashboards (Admin, Manager, Employee, Vendor)

## 🛠️ Tech Stack
- Frontend: React / Vite
- Backend: Spring Boot (Java)
- Database: MySQL

## ▶️ How to Run

### Backend
1. Open Backend in Eclipse/IDE
2. Run Spring Boot application

### Frontend
```bash
npm install
npm run dev
